# Todo List App 📝

A simple and elegant Todo List application built with React and Redux.  
Users can add, complete, undo, and delete tasks easily.

## 🚀 Features

- Add new todos
- Mark todos as completed
- Undo completed todos
- Delete todos
- Nice UI with SCSS

## 📦 Tech Stack

- React
- Redux
- SCSS (with BEM naming)
- React Icons
